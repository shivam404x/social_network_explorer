🚀 Social Network Explorer (SNE)

A simplified social network system built using core Data Structures concepts like Hashing, Graphs, BFS, DFS, and Sorting.

---

📌 Features

- 👤 Add users with interests
- 🤝 Create and remove friendships
- 🔍 Find shortest path using BFS (degrees of connection)
- 🌐 Explore network using DFS (friends-of-friends)
- ⭐ Recommend users based on common interests

---

🧠 Concepts Used

- Hashing → Dictionary ("dict")
- Graph → Adjacency List
- BFS → Shortest Path
- DFS → Depth-based traversal
- Sorting → Recommendation ranking
