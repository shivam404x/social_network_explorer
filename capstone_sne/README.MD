# Social Network Explorer (SNE)

This project is a simplified social network system using:
- Hashing (Dictionary)
- Graph (Adjacency List)
- BFS (Shortest Path)
- DFS (Exploration)
- Sorting (Recommendations)

## Features
- Add users & interests
- Add/remove friendships
- BFS shortest path
- DFS exploration
- Friend recommendation system

## Run
```bash
python sne.py
